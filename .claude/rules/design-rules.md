# AKKODiS Design Rules

## ブランドカラー（これ以外の色は使用禁止）
- Navy:  #001f33（および20/40/60/80%トーン）
- Gold:  #ffb81c（および20/40/60/80%トーン）
- Cyan:  #00ffff（および20/40/60/80%トーン）
- Black: #000000
- White: #ffffff

## 形状ルール
- 長方形に border-radius は使わない（円形の border-radius: 50% はOK）
- 色付き図形に色付き枠線はNG（例: 白ボックスに黄色ボーダーはNG）
- 広い背景に黄色・シアンはNG → 背景はネイビー or 白、黄色・シアンはアクセントのみ

## セッション終了時
- index.html の WHAT'S NEW を必ず更新すること
