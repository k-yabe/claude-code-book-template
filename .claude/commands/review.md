## コンテキスト

ステアリングファイル一覧:
```
!ls steering/S*.md 2>/dev/null | sort
```

## タスク

以下の手順でレビュー依頼フォームを出力してください。

1. `steering/` 内の全 `S*.md` ファイルを読み込み、**ステータスが「進行中」** のファイルを抽出する
2. 進行中のファイルが見つかった場合、各ファイルから以下を抜き出して整形する:
   - ファイル名・S番号
   - 目的セクション
   - スコープ（やること / やらないこと）
   - 完了条件
3. 以下のフォーマットで出力する:

---

## レビュー依頼

### 対象: [S番号] タイトル

**目的**
> （目的セクションの内容）

**スコープ（やること）**
- （作業項目一覧）

**スコープ外**
- （スコープ外項目）

**完了条件**
- （完了条件一覧）

---

承認してよいですか？

- **承認** → 「承認します」と返信してください。実装を開始します。
- **修正依頼** → 修正したい箇所を具体的に指示してください。

---

4. 進行中のファイルが存在しない場合:「進行中のステアリングファイルはありません。`/new-spec <タイトル>` で新しい作業を開始してください。」と出力する
