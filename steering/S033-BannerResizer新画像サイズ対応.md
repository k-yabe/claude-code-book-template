## 目的

ウェブサイト画像サイズ変更ガイド（2026/3/26）に基づき、Banner Resizerのプリセットサイズと機能を更新する。

---

## スコープ

### やること
- [x] バナー（MV）サイズを 800×446 → 1920×1080 に変更
- [x] 一覧（index）350×200 プリセットを削除（MVと同じ画像を使用するため）
- [x] 推奨アップロードサイズを 1200×669px → 1920×1080px に更新
- [x] 説明文を「3サイズ」→「2サイズ」に更新
- [x] サムネイル余白ガイド表示を追加（上下24px・左右100pxの安全ゾーンをオーバーレイ）
- [x] WHAT'S NEW を更新

### やらないこと（スコープ外）
- 正方形サイズ（1080×1080）のプリセット追加（カスタムサイズで対応可能）
- 400×400, 800×800 モジュール対応（グローバル側が制作中）

---

## 完了条件

- [x] OGP 1200×630 と MV 1920×1080 の2プリセットで動作する
- [x] MVプレビューに安全ゾーン枠（上下24px・左右100px）がオーバーレイ表示される
- [x] 推奨アップロードサイズが 1920×1080px になっている
- [x] 既存のカスタムサイズ・ZIP・AIアップスケール機能が引き続き動作する
