# S049 — 全アプリUI統一デザインシステム

## ステータス: 🔄 進行中

## 概要
全アプリのUI/デザインをプロレベルに統一する。
現状は各アプリが独自にCSSを定義しており、ボタンサイズ・余白・色・フォントサイズがバラバラ。

## 方針
1. `assets/app-styles.css` をCSS変数ベースの包括的デザインシステムに拡張
2. 共通コンポーネント（navbar, container, button, card, form, footer, loading）を共通CSSに集約
3. 各アプリのinline CSSから重複定義を削除し、共通CSSを参照
4. アプリ固有のスタイルのみinlineに残す
