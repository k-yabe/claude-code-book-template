# [S043] Prompt Maker v8 UIポリッシュ

**ステータス**: 進行中
**作成日**: 2026-03-31
**完了日**: —

---

## 目的

Prompt MakerのUI品質をNotebookLM/ChatGPT級に引き上げる。ローディング体験・マイクロインタラクション・ビジュアルフィードバック・ブランド準拠を一括改善。

---

## スコープ

### 1. ローディング体験の強化
- ソース要約生成中にシマーアニメーション（骨格UI）
- PDF解析中にプログレスバー表示
- AI生成ボタンにスピナー＋テキスト変化
- ローディングドットを大きく＋視認性向上

### 2. トースト通知
- インライン状態表示をスライドイン式トーストに置換
- 成功/エラー/情報の3種
- 4秒で自動消去

### 3. モーダルアニメーション
- 背景ブラー + フェードイン
- ESCキーで全モーダル閉じる
- コンテンツのスケールアニメーション

### 4. ボタン・インタラクション統一
- Primary/Secondary/Ghostの3バリアント
- hover時のlift（shadow）効果統一
- disabled状態の改善（cursor: not-allowed）

### 5. ブランドカラー準拠修正
- `#00b894`（緑）→ `#ffb81c`（ゴールド）
- チャットアバター背景をブランドカラーに
- ソースバッジをネイビー背景に

### 6. 検索UX改善
- 検索結果件数表示
- クリアボタン（×）
- マッチテキストハイライト

### 7. 品質スコア可視化改善
- バー太さ12px＋パーセンテージラベル
- 軸名のレイアウト改善

### 8. 引用マーカー視認性改善
- フォントサイズ拡大
- ホバー時のリフト効果
- ハイライト持続時間延長
