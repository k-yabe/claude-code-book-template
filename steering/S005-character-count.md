# [S005] 文字数カウントアプリ

**ステータス**: 完了
**作成日**: 2026-02-25
**完了日**: 2026-02-25

---

## 目的

テキストを貼り付けるだけで文字数・単語数をリアルタイムに確認できる、マーケター向けのシンプルなWebアプリを作る。

---

## スコープ

### やること
- [ ] `counter.html` を1ファイルで新規作成（HTML + CSS + JS）
- [ ] テキストエリアへの入力をリアルタイムにカウント
  - 文字数（スペース含む）
  - 文字数（スペース除く）
  - 単語数
  - 行数
- [ ] よく使う文字数制限の目安を表示（超えたら警告色）
  - メール件名：35文字
  - X（旧Twitter）：140文字
  - メタディスクリプション：160文字

### やらないこと（スコープ外）
- サーバーへの保存・ログイン機能
- PDF・Word ファイルの読み込み
- 多言語対応（英語カウントの特別処理）

---

## 完了条件

- [ ] テキストを入力するとリアルタイムでカウントが更新される
- [ ] 制限を超えたら該当項目が赤くなる
- [ ] スマホでも見やすいレイアウト
