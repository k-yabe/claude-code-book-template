# [S043] BannerResizer ガイド準拠の容量制限追加

**ステータス**: 完了 <!-- 未着手 | 進行中 | レビュー中 | 完了 -->
**作成日**: 2026-03-31
**完了日**: 2026-04-02

---

## 目的

「Webサイト画像サイズ 変更ガイド」に準拠し、Banner Resizerにダウンロード画像の容量制限チェック・自動圧縮機能を追加する。MV画像は1M以内、OGP画像は100KB以下の要件を満たす。

---

## スコープ

### やること
- [x] MV（バナー）画像のダウンロード時に1MB以内になるよう品質調整する
- [x] OGP画像のダウンロード時に100KB以下になるよう品質調整する（PNG固定のため超過時は警告表示）
- [x] 容量がガイド要件を超える場合、自動的にクオリティを下げて再圧縮する（lossy形式のみ）
- [x] プレビューカード上にファイルサイズの目安を表示する
- [x] JPG出力オプションを廃止（PNG/WebPのみに統一）

### やらないこと（スコープ外）
- 正方形画像（1080×1080）の追加
- ブログ本文画像（400×400/800×800）の追加
- 安全ゾーンのUI変更

---

## 完了条件

- [x] MV画像のWebPダウンロードが1MB以内に収まること
- [x] OGP画像のPNGダウンロードが100KB以下に収まること（超過時は警告表示）
- [x] 容量超過時に自動圧縮が動作すること（lossy形式のみ）
- [x] プレビューにファイルサイズ目安が表示されること
- [x] JPG出力オプションが廃止されていること
