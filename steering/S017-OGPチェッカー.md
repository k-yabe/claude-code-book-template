## 目的

URLを入力するだけでOGPタグを解析し、Twitter/X・Facebook・LINEでの表示プレビューを確認できるツールを提供する。マーケターがSNSシェア前にOGP設定の不備を手軽にチェックできるようにする。

---

## スコープ

### やること
- [x] URL入力フォームの実装
- [x] CORSプロキシ（allorigins.win）経由でHTMLを取得・OGPタグを解析
- [x] `og:title` / `og:description` / `og:image` / `og:type` / `og:url` の表示
- [x] Twitter/X カードプレビュー（PC）
- [x] Facebook シェアプレビュー（PC）
- [x] LINE シェアプレビュー
- [x] バリデーション表示（必須タグ欠損・文字数超過の警告）
- [x] `ogp-checker.html` として新規ファイル追加

### やらないこと（スコープ外）
- 画像アップロード機能
- はてなブックマークプレビュー
- スマホ/アプリ別プレビュー
- 複数URL同時比較

---

## 完了条件

- [x] URLを入力してOGPタグが正しく取得・表示される
- [x] Twitter/X・Facebook・LINEの3種プレビューが表示される
- [x] og:title / og:description / og:image が欠損している場合に警告が出る
- [x] ブラウザで直接開いて動作する（ビルド不要）
